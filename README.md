進階版：JS 幸運連線拉霸機 (Slot Machine)
除了基礎的 Python 命令行遊戲外，本專案新增了具備視覺互動介面的 Web 版拉霸機遊戲。

✨ 新增功能特色
視覺化老虎機介面：使用 CSS 打造三個獨立的數字轉輪區塊（Slot Boxes），提供更直覺的遊戲體驗。

隨機連線邏輯：按下「開始轉動」按鈕後，系統會同步產生三個 1-5 之間的隨機數。

自動中獎偵測：

中獎條件：當三個數字完全相同時（例如 7-7-7），觸發中獎機制。

即時反饋：中獎時背景會自動變為綠色並顯示恭喜訊息；失敗時則顯示紅色提示。

嘗試次數紀錄：系統會自動累計玩家點擊的次數，增加遊戲的挑戰性。

🛠️ 技術實作亮點
DOM 操作：透過 document.getElementById 即時抓取並更新網頁元素內容。

事件監聽：運用 addEventListener 讓按鈕具備觸發隨機邏輯的能力。

條件判斷：使用 JavaScript 的 if...else 邏輯與 && 運算子判定連線成功與否。
